== KApp Controller Install

Install `kapp-controller`:

Deploy controller:
[source, bash]
----
kapp deploy \
  -a kc \
  -y \
  -f https://github.com/vmware-tanzu/carvel-kapp-controller/releases/download/v0.28.0/release.yml
----
